编译版本要求

请使用版本

targetSdkVersion 23

关于baseactivity

  • 使用mvp框架界面，请继承BaseActivity
  • 使用其它界面，继承HetBaseActivity

初始化配置

请在application里面加上以下配置

    AppGlobalHost.setHost(AppGlobalHost.TEST_HOST); //设置请求地址
    AppDelegate.init(this);
    AppDelegate.setAppId(APP_ID_VALUE);   //设置appid
    AppDelegate.setAppSecret(APP_SECRET);
    HetLoginSDKDelegate.init(this);//改行代码必须在AppDelegate.initActiveAndroid()之前调用
    HetH5SdkManager.getInstance(this).inith5Sdk(appSign);  //h5初始化
    HetBindUiSdkManager.getInstance().init();//该行代码必须在AppDelegate.initActiveAndroid()之前调用
    HetFriendSDKDelegate.init(this);   //好友模块配置
    AppDelegate.initActiveAndroid();   //数据库初始化
    mLoginDelegate = new HetThirdDelegate.Builder(this) //登录模块配置
            .registerQQ(mQQAppID,mQQAppSecret)
            .registerWeixin(mWXAppID,mWXAppSecret)
            .registerSinaWeibo(mSinaAppID,mSinaAppSecret,mSinaRedirectURL)
            .create();
